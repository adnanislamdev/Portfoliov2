## Personal Portfolio Website

This is a personal portfolio website built with Next.js and React. The website displays information about me and showcases my projects.

## Website Overview

The site has two pages accessible through the navigation:

- **Home**: Introduction and about-me section covering my background, education, and experience.
- **Projects**: Showcases my portfolio projects with descriptions, technologies used, and links to live demos or GitHub repositories.

A contact form backed by a database will be added back in a future update.
